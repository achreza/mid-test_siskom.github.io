Langkah Langkah Instalasi Sistem Operasi Open source distribusi bebas

1. Pertama tama download virtual box pada web Resminya, lalu pillih sesuai platform anda



2. Setelah mendownloa d file yang tadi, Langkah selanjutnya yakni kita membuka file installer VirtualBoxny a, klik next saja terus sampai proses instalasi berjalan.



3. Setelah Instalasi VirtualBox Berhasil, sekarang kita buka aplikasinya



4. Setelah kita mengklik tombol new di menu utama VirtualBox, selanjutnya kita membuat virtual machine dengan memilih



5. Dimenu ini kita mengatur memori yang dialokasikan untuk Virtual Machine yang akan kita buat



6. Diproses ini kita harus menentukan haddisk kita nanti di Virtual Machine, disini saya membuat Virtual Disk baru,jika sudah kita klik create



7. Setelah membuat sebuah virtual machine baru, di menu utama kitab isa pilih virtual machine nya, lalu kita klik Start.



8. Setelah memilih file disk tadi, kita bisa melakukan booting pada virtual machine, dengan menekan tombol start pada menu ini



9. Dimenu ini kita diarahkan langsung untuk menginstall ubuntu, bis akita pilih tombol Install Ubuntu



10. Dimenu ini kita disuruh untuk memilih Bahasa untuk layout keyboard yang nantinya akan digunakan di distro linux



11. Disini kita disuruh untuk memilih tipe instalasi yang akan dilakukan, disini saya memilih minimal installation (tanpa menginstal beberapa software)



12. Disini ki disuruh memilih lokasi partisi untuk kita menginstall operasi system ini



13. Dimenu ini kita disuruh menentukan dimana lokasi kita



14. Dimenu ini kita disuruh untuk menginput username, user computer name, dan juga password



15. Disini kita hanya perlu menunggu sampai proses instalasi selesa



16. Disini kita diarahkan untuk merestart pc kita karena sudah berhasil menginstal Ubuntu



17. Kini Distro linux ubuntu sudah siap kita gunakan.

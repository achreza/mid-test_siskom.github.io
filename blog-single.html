Langkah Langkah Instalasi Sistem Operasi Proprietary

1. Pertama-tama download virtual box pada web Resminya, lalu pillih sesuai platform anda



2. Setelah mendownload file yang tadi, Langkah selanjutnya yakni kita membuka file installer VirtualBoxnya, klik next saja terus sampai proses instalasi berjalan.



3. Setelah Instalasi VirtualBox Berhasil, sekarang kita buka aplikasinya



4. Setelah kita mengklik tombol new di menu utama VirtualBox, selanjutnya kita membuat virtual machine dengan memilih



5. Dimenu ini kita mengatur memori yang dialokasikan untuk Virtual Machine yang akan kita buat



6. Diproses ini kita harus menentukan haddisk kita nanti di Virtual Machine, disini saya membuat Virtual Disk baru,jika sudah kita klik create



7. Langkah selanjutnya kita pilih virtual machine kita, lalu kita klik tombol start



8. Saat melakukan pertama kali start pada virtual machine, kita akan disuruh untuk memilih start up disk. Untuk memilihnya kita bisa klik tombol folder



9. Di menu ini, kitab isa mengklik tombol add, kaku pilih iso file OS yang akan kita install di folder kita, Lalu pilih.



10. Setelah memilih file ISO tadi, Langkah selanjutnya yakni kita tekan tombol ‘Choose’



11. Setelah memilih file disk tadi, kita bisa melakukan booting pada virtual machine, dengan menekan tombol start pada menu ini



12. Pada menu ini, kita disurug memilih Bahasa, waktu, input method untuk pc kita nantinya.



13. Di menu ini kita akan memilih system operasi apa yang akan kita install nantinya.



14. Dimenu ini kitab isa mengklik centang ‘I accept the license terms’, dan mengklik next



15. Dimenu ini kita bisa memilih upgrade atau custom, disini saya memilih custom karena belum ada os sebelumnya di Virtual machine ini.



16. Dimenu ini kita memilih partisi mana yang akan kita install windowsnya, dimenu ini juga kitab isa memformat partisi yang ada



17. Dimenu ini bisa dilihat bahwa proses instalasi windows sudah berjalan, kitab isa menunggu sampai selesai



18. Dimenu ini kita akan diarahkan untuk melakukan pengaturan awal di windows, bisa kita klik ‘use express setting’ untuk lebih cepatnya.



19. Dimenu ini kita diminta untuk menginput nama user dan password untuk keamanan user di windows.



20. Windows sudah siap digunakan.
